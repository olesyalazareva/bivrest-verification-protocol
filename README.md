# BIVREST — метод игровой деконструкции сценариев

[English version](./README.en.md)


**Where to get ground truth data for emotion recognition and deception detection — without manual labeling.**

This is a ready-to-use protocol for generating behavioral datasets through a structured game with binary "Believe / Don't Believe" verification.

**For:**
- AI/ML engineers who need annotated datasets for affective computing
- R&D teams looking for unbiased assessment tools
- Researchers studying emotional authenticity and defense mechanisms

Includes: full protocol, annotation guide, and raw session logs.


**Где взять эталонные данные для распознавания эмоций и детекции лжи — без ручной разметки.**

Это готовый протокол для генерации поведенческих датасетов через ролевую игру с бинарной верификацией «Верю / Не верю».

**Для:**
- AI/ML-инженеров, которым нужны размеченные данные для аффективных вычислений
- R&D-отделов, ищущих инструменты объективной оценки
- Исследователей эмоциональной подлинности и защитных механизмов

Включены: полный протокол, гайд по аннотации, сырые логи сессий.

## Структура репозитория

- **`/protocol`** — полное описание метода BIVREST (7 шагов, 7 отличий, псевдокод, API reference).
- **`/game`** — игра «Вселенная 69» (правила, карты, аватары).
- **`/annotation-guide`** — гайд по аннотации данных для AI.
- **`/logs`** — пример сырого лога с сессии.

---

## Как использовать метод

1. Изучите `protocol/BIVREST_method.md`.
2. Организуйте пилотные сессии по протоколу.
3. Собирайте логи действий и голосований «Верю / Не верю».
4. Обучайте модель на этих данных.

## 🎮 Связь с игрой UNIVERSE 69

BIVREST и **UNIVERSE 69** — это два взаимодополняющих проекта, созданных для работы с AI и психологическими исследованиями:

| **BIVREST** | **UNIVERSE 69** |
|:---|:---|
| Генератор ground truth данных для AI | LLM-native движок для ролевых игр |
| Обучение детекции правды/лжи и эмоций | Борьба с дрейфом контекста и галлюцинациями |
| Протокол с бинарной верификацией | Карточная механика UNO как структурный контейнер |
| Для AI/ML-инженеров и исследователей | Для геймдизайнеров, психологов и разработчиков |

**Как использовать вместе:**
1. Используйте **UNIVERSE 69** для проведения ролевых сессий с LLM
2. Применяйте протокол **BIVREST** для верификации и сбора данных
3. Обучайте модели на размеченных данных, полученных из игры
