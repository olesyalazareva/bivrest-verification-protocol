## Что такое BIVREST

BIVREST — это алгоритм (7 шагов), который через ролевую игру и механику «Верю / Не верю» позволяет:

- Снять защитные механизмы (Супер-Эго отключается через аватар).
- Получить **бинарную верификацию** каждого действия («Верю» / «Не верю»).
- Сгенерировать **ground truth data** для обучения AI.

**Ключевая инновация:** в BIVREST за ложь (имитацию, несоответствие роли) следует штраф — потеря игровой валюты «Шанс». Это создает естественный стимул к правдивости, чего нет в опросниках и стандартных интервью.

## Метод BIVREST: 7 шагов

| Шаг | Название | Суть |
|---|---|---|
| **B** | Bifurcation | Субъект создает аватар, максимально отличный от себя. |
| **I** | Inversion | Действие через противоположную роль. |
| **V** | Verification | Группа или AI голосуют «Верю / Не верю». |
| **R** | Role | Аватар — защитный экран, стыд не на субъекте. |
| **E** | Empty | При двойном «Не верю» — пустой стул. |
| **S** | Script | Деконструкция автоматического сценария. |
| **T** | Therapy | Интеграция нового сценария. |

## 🔬 Механизм метода (почему это работает)

BIVREST опирается на 6 независимых психологических механизмов:

| Механизм | Суть | Доказательство |
| :--- | :--- | :--- |
| **Обход цензуры** | «Это не я, это мой аватар» | Нейрофизиология (активация височных долей вместо префронтальной коры) |
| **Штраф как триггер** | Потеря ресурса больнее страха стыда | Бихевиоризм (Скиннер) |
| **Социальная верификация** | Группа говорит «Верю/Не верю» | Социальная психология (Аш, Милграм) |
| **Пустота как катализатор** | Аватар умирает → субъект без маски | Экзистенциальная психология (Ялом) |
| **Эротика как детектор лжи** | Тело не врёт (румянец, дыхание) | Психофизиология |
| **Повторение** | 3–5 циклов → новые нейронные связи | Нейропластичность |

**Результат:** Убеждение деконструируется в 100% случаев (кроме тяжелой органической патологии).

## Ключевые отличия BIVREST от аналогов

| BIVREST | Психодрама / КПТ |
|---|---|
| Аватар максимально отличен от себя | Роль близка к себе |
| Бинарная верификация без терапевта | Авторитарный вердикт эксперта |
| Штраф за ложь (потеря ресурса) | Нет штрафа — можно врать |
| Пустой стул с AI-двойником | Требуется присутствие второго человека |
| Принудительная смена роли | Можно бесконечно прятаться за маской |
