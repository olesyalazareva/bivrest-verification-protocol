## Что такое BIVREST

BIVREST — это алгоритм (7 шагов), который через ролевую игру и механику «Верю / Не верю» позволяет:

- Снять защитные механизмы (Супер-Эго отключается через аватар).
- Получить **бинарную верификацию** каждого действия («Верю» / «Не верю»).
- Сгенерировать **ground truth data** для обучения AI.

**Ключевая инновация:** за ложь (имитацию, несоответствие роли) следует штраф — потеря игровой валюты «Шанс».

## Метод BIVREST: 7 шагов

| Шаг | Название | Суть |
|-----|----------|------|
| **B** | Bifurcation | Создание аватара, максимально отличного от себя |
| **I** | Inversion | Действие через противоположную роль |
| **V** | Verification | Голосование «Верю / Не верю» |
| **R** | Role | Аватар как защитный экран |
| **E** | Empty | При двойном «Не верю» — пустой стул |
| **S** | Script | Деконструкция автоматического сценария |
| **T** | Therapy | Интеграция нового сценария |

## Для AI / ML специалистов

### Что вы получаете

1. **Протокол сбора данных** с бинарной разметкой
2. **Гайд по аннотации** для обучения AI-верификатора
3. **Пример данных** — реальный лог сессии

### Как использовать

1. Изучите `protocol/BIVREST_method.md`
2. Организуйте пилотные сессии по протоколу
3. Собирайте логи действий и голосований
4. Обучайте модель на этих данных

## 🔬 Почему это работает

| Механизм | Суть |
|----------|------|
| Обход цензуры | «Это не я, это мой аватар» |
| Штраф как триггер | Потеря ресурса больнее страха стыда |
| Социальная верификация | Группа говорит «Верю/Не верю» |
| Пустота как катализатор | Аватар умирает → субъект без маски |
| Повторение | 3–5 циклов → новые нейронные связи |
