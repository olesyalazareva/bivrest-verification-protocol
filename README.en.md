# BIVREST — Ground Truth Generator for AI

**Behavioral Verification through Role-Play and Binary Assessment**

[![License: Custom](https://img.shields.io/badge/License-NonCommercial--with--attribution-blue.svg)](./LICENSE)
[![Status](https://img.shields.io/badge/Status-Deposited%20(NRIS)-green)](./LICENSE)

**BIVREST** is a ready-to-use protocol for generating high-quality ground truth datasets for affective computing, emotion recognition, and deception detection—**without costly manual labeling**.

It uses a structured role-playing game with a binary "Believe / Don't Believe" mechanic and a penalty for imitation, forcing authentic behavioral responses. The result is a rich, annotated dataset of emotional and behavioral patterns.

---

## 🎯 Who is it for?

- **AI/ML Engineers** needing annotated datasets for training models in affective computing and deception detection.
- **R&D Teams** looking for unbiased, game-based assessment tools.
- **Researchers** studying emotional authenticity, defense mechanisms, and human-AI interaction.

---

## ✨ Key Features

- **Structured 7-Step Protocol:** A clear, repeatable methodology for data collection.
- **Binary Verification:** Uses "Believe / Don't Believe" voting, eliminating subjective expert bias.
- **Authenticity Incentive:** A built-in penalty for imitation ("fake" behavior) forces genuine emotional responses.
- **Role-Play Framework:** Uses the "Universe 69" game with avatars to create psychological safety and distance.
- **Complete Toolkit:** Includes protocol, annotation guide, and raw session logs.
- **API Reference:** For seamless integration into your data pipeline.

## 🆚 How BIVREST Compares to Alternatives

| **BIVREST** | **Psychodrama / CBT** |
|:---|:---|
| Avatar is maximally different from self | Role is close to self |
| Binary verification without a therapist | Authoritative verdict by an expert |
| Penalty for lying (resource loss) | No penalty—safe to lie |
| Uses an AI "double" for the empty chair | Requires the presence of a second person |
| Enforces role switching | Can hide behind a mask indefinitely |

| **BIVREST** | **UNIVERSE 69** |

## 🚀 How to Use the Method

1. **Study** the full protocol in `protocol/BIVREST_method.md`.
2. **Organize** pilot sessions following the structured game rules.
3. **Collect** raw action logs and "Believe / Don't Believe" votes from participants.
4. **Annotate** the data using the guide in `/annotation-guide`.
5. **Train** your AI models on this behavioral dataset.
